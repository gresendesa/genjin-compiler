## Contexto

O proc-bloco `Troca_Ferramenta` em `examples/lenhador-sintetico.gnj` está implementado com
`exec` canônicos aninhados em 6 níveis, resultando em recuo excessivo (~70 colunas) que
prejudica legibilidade. A notação inline (`@proc() when(CODE) while(CODE)`) foi criada
exatamente para achatar esse padrão de encadeamento de sucesso.

---

## Escopo

Refatorar o corpo de `Troca_Ferramenta` para usar a notação inline, transformando a cadeia
de `exec` canônicos aninhados em uma `InlineSeqNode` com átomos encadeados via `when(SUCESSO)`.

### Estrutura atual (canônica, 6 níveis de recuo):

```gnj
Troca_Ferramenta(home_gastos: &Text, home_novos: &Text, home_voltar: &Text) {
    exec Definir_Home(home=&home_voltar) as "Troca:Definir_Home" {
        case SUCESSO:
            exec Teleportar(home=&home_gastos) as "Troca:TP_Gastos" {
                case SUCESSO:
                    exec Guardar_Itens_MKB(...) as "Troca:Guardar" {
                        case SUCESSO:
                            exec Teleportar(home=&home_novos) as "Troca:TP_Novos" {
                                case SUCESSO:
                                    exec Pegar_Itens_MKB(...) as "Troca:Pegar" {
                                        case SUCESSO:
                                            exec Teleportar(home=&home_voltar) as "Troca:Voltar" {
                                                pass SUCESSO, NÃO_EXISTE, CANCELADO
                                            } while(DEMORA)
                                    } while(DEMORA)
                                pass NÃO_EXISTE, CANCELADO
                            } while(DEMORA)
                    } while(DEMORA)
                pass NÃO_EXISTE, CANCELADO
            } while(DEMORA)
        pass SUCESSO, NÃO_EXISTE, REINICIAR_CICLO, LIMITE_ALCANCADO, PROIBIDO
    } while(DEMORA, CANCELADO)
}
```

### Estrutura alvo (inline achatada):

```gnj
Troca_Ferramenta(home_gastos: &Text, home_novos: &Text, home_voltar: &Text) {
    @Definir_Home(home=&home_voltar) when(SUCESSO) while(DEMORA, CANCELADO)
    @Teleportar(home=&home_gastos) when(SUCESSO) while(DEMORA)
    @Guardar_Itens_MKB(gaveta="gaveta_mkb_guardar", itens="['diamond_axe']", quantidade=1, quantidade_movidos_var=&quantidade_movidos) when(SUCESSO) while(DEMORA)
    @Teleportar(home=&home_novos) when(SUCESSO) while(DEMORA)
    @Pegar_Itens_MKB(gaveta="gaveta_mkb_pegar", itens="['diamond_axe']", quantidade=1, quantidade_movidos_var=&quantidade_movidos) when(SUCESSO) while(DEMORA)
    @Teleportar(home=&home_voltar)
}
```

> conforme o padrão DoD da SPR-2026-12 (S12-T15).
