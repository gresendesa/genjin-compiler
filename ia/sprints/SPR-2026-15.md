## Causa Raiz (análise técnica)

### Fluxo reproduzido por `examples/test.gnj`

```
doe (LOOP_WHILE=[], PASS_CODES=['ERROR', 'OK'])
  case OK →
    xpto (LOOP_WHILE=[], PASS_CODES=['ERROR'])
      case OK →
        baz (LOOP_WHILE=['ERROR', 'DEMORA'], PASS_CODES=[])
          case OK →
            bar (LOOP_WHILE=[], PASS_CODES=['OK', 'ERROR', 'DEMORA'])
```

### Rastreamento do bug em `get_unhandled_codes_recursively`

A função é chamada recursivamente para coletar todos os códigos "não tratados" da subárvore,
armazenando-os no `collection_dict` **compartilhado**.

1. Entra em `doe` → chama recursão para `xpto` → chama para `baz` → chama para `bar`.
2. `get_unhandled_codes(bar)` retorna `['OK', 'ERROR', 'DEMORA']` (bar não trata nenhum localmente).
3. Esses três códigos são adicionados a `collection_dict['status_var']`.
4. A recursão retorna para `baz`. `baz` tem `LOOP_WHILE=['ERROR', 'DEMORA']`.
   **O problema:** nenhum código é removido de `collection_dict['status_var']` após a recursão
   dos filhos, mesmo que `baz` os absorva via `LOOP_WHILE`.
5. A recursão retorna para `doe`. `doe` encontra `'DEMORA'` em `unhandled_codes` e,
   como 'DEMORA' não está em `PASS_CODES=['ERROR', 'OK']`, lança a exceção.

### Fix identificado

Em `get_unhandled_codes_recursively`, ao processar os casos filhos de um bloco, aplicar o
filtro do `LOOP_WHILE` do bloco atual sobre os **novos** códigos adicionados pelos filhos
(usando snapshot antes/depois):

```jinja2
{# Snapshot antes de processar filhos #}
{* set snapshot_before = collection_dict[variable_name] | list *}

{* for case in block[ATTRIBUTE.CASES] *}
    ... (recursão existente)
{* endfor *}

{# Filtrar códigos de filhos absorvidos pelo LOOP_WHILE do bloco atual #}
{* if block[ATTRIBUTE.LOOP_WHILE] is iterable and block[ATTRIBUTE.LOOP_WHILE] is not string and block[ATTRIBUTE.LOOP_WHILE] | length > 0 *}
    {* set from_children = collection_dict[variable_name] | reject('in', snapshot_before) | list *}
    {* set surviving = from_children | reject('in', block[ATTRIBUTE.LOOP_WHILE]) | list *}
    {* set _ = collection_dict.update({variable_name: snapshot_before + surviving}) *}
{* endif *}
```
