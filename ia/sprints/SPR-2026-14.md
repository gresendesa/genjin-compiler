## Detalhes técnicos para implementação

### S14-T01 + S14-T02: Mudança no parser

**Localização exata:**
- `InlineAtomNode` — `compiler/parser.py` linha 83 (campo `while_code`)
- `_parse_inline_atom` — `compiler/parser.py` linha ~404 (bloco `if self._check(TokenType.KW_WHILE)`)

**Código atual (linhas ~404–421):**
```python
while_code: str | None = None
if self._check(TokenType.KW_WHILE):
    self._advance()  # while
    self._expect(TokenType.LPAREN, "esperado '(' após 'while'")
    code_tok = self._expect(TokenType.IDENT, "esperado código no while")
    if code_tok.value not in valid_codes:
        raise ParseError(
            f"código '{code_tok.value}' em 'while' não declarado no proc '{proc_name}'",
            code_tok.line,
        )
    self._expect(TokenType.RPAREN, "esperado ')' após código do while")
    while_code = code_tok.value
```

**Código alvo:**
```python
while_codes: list[str] = []
if self._check(TokenType.KW_WHILE):
    self._advance()  # while
    self._expect(TokenType.LPAREN, "esperado '(' após 'while'")
    while not self._check(TokenType.RPAREN, TokenType.EOF):
        if while_codes:
            self._expect(TokenType.COMMA, "esperado ',' entre códigos do while")
        while_codes.append(self._expect(TokenType.IDENT, "esperado código no while").value)
        # SEM validação contra valid_codes — borbulhamento é válido
    self._expect(TokenType.RPAREN, "esperado ')' após códigos do while")
```

**Nota:** `when(CODE)` tem a mesma restrição (valida contra `valid_codes`). Avaliar se deve
ser corrigido junto. Se `when` também borbulha, corrigir; caso contrário, deixar por ora.

**Alteração em `InlineAtomNode` (linha 89):**
```python
# Antes:
while_code: str | None      # de while(CODE)

# Depois:
while_codes: list[str]      # de while(CODE, CODE, ...)
```

**Atualização no `return` (linha ~433):**
```python
# Antes:
while_code=while_code,

# Depois:
while_codes=while_codes,
```

### S14-T06: Estrutura alvo de `Troca_Ferramenta`

```gnj
Troca_Ferramenta(home_gastos: &Text, home_novos: &Text, home_voltar: &Text) {
    @Definir_Home(home=&home_voltar) when(SUCESSO) while(DEMORA, CANCELADO)
    @Teleportar(home=&home_gastos) when(SUCESSO) while(DEMORA)
    @Guardar_Itens_MKB(gaveta="gaveta_mkb_guardar", itens="['diamond_axe']", quantidade=1, quantidade_movidos_var=&quantidade_movidos) when(SUCESSO) while(DEMORA)
    @Teleportar(home=&home_novos) when(SUCESSO) while(DEMORA)
    @Pegar_Itens_MKB(gaveta="gaveta_mkb_pegar", itens="['diamond_axe']", quantidade=1, quantidade_movidos_var=&quantidade_movidos) when(SUCESSO) while(DEMORA)
    @Teleportar(home=&home_voltar)
}
```

**Nota:** O primeiro átomo usa `while(DEMORA, CANCELADO)` — esse é exatamente o caso que
requer B-024 (múltiplos códigos no `while` inline).
