## Escopo

Transformar o compilador Genjin em um pacote pip instalável (`genjin-compiler`) com três pontos de extensão customizáveis:

1. **`GnjSourceResolver`** — resolve source de arquivos `.gnj` para imports (hoje: filesystem)
2. **`TemplateLoader`** — fornece Jinja2 loader para o assembler (hoje: `DottedLoader(FileSystemLoader)`)
3. **`GnjTemplateName`** — nome do template base emitido pelo transpiler (hoje: hardcoded `"genjin"`)

API pública: `CompilerConfig` (dataclass) com campos opcionais para os 3 pontos.
